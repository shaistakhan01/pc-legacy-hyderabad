# PC Legacy Hyderabad — Hotel Management & Booking Web Application

A modern hotel management and booking platform covering Room Booking,
Restaurant & Table Reservation, Banquet & Event Booking, Meetings &
Conference Booking, a Customer Portal, and an Admin Portal.

**Stack:** React.js + TypeScript (frontend) · Node.js + Express.js + TypeScript (backend) · Supabase (Database, Auth, Storage, Realtime)
